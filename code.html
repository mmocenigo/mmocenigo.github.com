Baby Name Data ( C )

Written in C, this is a simple program which uses US Census data files to create histograms of the frequency of baby names. The code includes a make file and the data files for a little over 11 decades of names. The executable will be called "babynames". To run the program, type "./babynames -h [NAME] [FILES]". For instance, "names*" will include all the name files.

Spreadsheet ( Java )

This program is very basic, but it is an exploration of the Java GUI. The user specifies the rows and columns for a spreadsheet, and the program creates a spreadsheet that allows string or number input. With numerical input, the cell data can be added, subtracted, multipled, or divided with the data of another cell.
